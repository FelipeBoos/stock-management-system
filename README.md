# Stock Management System

Java console application for inventory and order management.

## Tech Stack
- Java 17
- Maven

## How to run
```bash
mvn clean compile
mvn exec:java

## Project Structure

- **app/** — Application entry point
- **domain/** — Domain entities
- **repository/** — Data access
- **service/** — Business rules
- **ui/** — Console menus
